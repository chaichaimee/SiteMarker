SiteMarker


Marca tu camino, salta con facilidad


autor: chai chaimee

Descripción

Cada día pasas horas dentro de los mismos sitios web – leyendo artículos, revisando paneles, rellenando formularios o navegando por largas listas. Para moverte de una sección importante a otra, pulsas las teclas de flecha una y otra vez, buscando ese párrafo o botón que necesitas. Es repetitivo, agotador y desperdicia tu valioso tiempo.

SiteMarker cambia eso. Te permite colocar marcadores permanentes en cualquier párrafo o elemento dentro de una página web. Una vez marcado, puedes saltar directamente a ese lugar con una sola pulsación de tecla – hacia adelante o hacia atrás, en cualquier orden. No más desplazamientos interminables ni maratones de flechas. Solo marca, salta y haz lo que tengas que hacer.


Teclas rápidas

La tecla principal para todas las acciones es Windows+F12. Esta es una tecla de pulsación múltiple – su comportamiento cambia dependiendo de cuántas veces la pulses rápidamente (en 0,4 segundos).

  • Pulsación única (Windows+F12 una vez) – Si estás en una línea que ya tiene un marcador, se abre el diálogo de edición para ese marcador. Si la línea no tiene marcador todavía, aparece el diálogo Añadir marcador, ya rellenado con el texto del párrafo actual.
  • Doble pulsación (Windows+F12 dos veces rápido) – Abre el Administrador de sitios para el sitio web actual. Si el sitio ya existe, puedes editar su configuración o gestionar todos sus marcadores. Si el sitio es nuevo, se te pedirá que lo añadas.

Primeros pasos – Paso a paso

1. Añade tu primer sitio

Cuando abras un sitio web donde quieras usar marcadores, pulsa dos veces Windows+F12. Aparece el diálogo Añadir sitio.

  • Nombre mostrado: Dale a tu sitio un nombre amigable (p. ej., “Bandeja de Gmail” o “Noticias diarias”).

  • Patrón de URL: Introduce la dirección web (o parte de ella) que identifique este sitio.

  • Tipo de coincidencia: Elige cómo SiteMarker debe comparar la URL actual con tu patrón.

    Solo dominio – Coincide solo con el dominio principal (p. ej., el patrón google.com funciona para www.google.com, mail.google.com, pero no para google.co.uk).

    Incluir subdominios – Como “Solo dominio”, pero la URL debe terminar con tu patrón. Por ejemplo, el patrón example.com coincide con sub.example.com y news.example.com, pero no con example.org.

    Contiene subcadena – Coincide si tu patrón aparece en cualquier parte de la URL completa (incluyendo ruta y consulta). Ejemplo: el patrón /search coincide con https://www.google.com/search?q=test.

    Coincidencia exacta – La URL completa debe ser exactamente igual a tu patrón (sin distinguir mayúsculas/minúsculas). Úsalo cuando quieras marcadores solo para una página específica, como https://example.com/dashboard.

    Expresión regular – Para usuarios avanzados. Escribe un patrón regex (p. ej., https://.*\.example\.com/.*) para que coincida con cualquier página bajo cualquier subdominio de example.com.

Haz clic en Guardar sitio. Tu sitio ya está registrado y puedes empezar a colocar marcadores.


2. Coloca tu primer marcador

Navega hasta el párrafo, línea o botón exacto que quieras recordar. Luego pulsa una vez Windows+F12. Se abre el diálogo Añadir marcador.

  • Patrón – SiteMarker rellena automáticamente este campo con el texto que tiene el foco actualmente. Puedes editarlo si es necesario. Este texto es lo que SiteMarker buscará.

  • Coincidencia de patrón – Elige cómo debe compararse el patrón:

    Contiene texto – Encuentra el patrón en cualquier lugar dentro del párrafo (coincidencia parcial).

    Párrafo exacto – El párrafo debe ser igual al patrón exactamente (coincidencia de texto completo).

    Coincidencia regex – Usa una expresión regular para coincidencias avanzadas.

  • Modo – Decide qué hace este marcador:

    Saltar – Cuando pulses la tecla de salto asignada, el cursor se mueve a este marcador. Puedes asignar una de cuatro teclas de salto: J, F, D o Z.
    Pulsa la tecla sola para saltar hacia adelante al siguiente marcador con esa tecla. Pulsa Mayús+tecla para saltar hacia atrás.

    Auto clic – En lugar de solo mover el foco, SiteMarker hará automáticamente un clic en el elemento cuando pulses la tecla de clic asignada. Las teclas de clic disponibles son Alt+J, Alt+C, Alt+X y Alt+Z.

  • Nombre mostrado – Dale a tu marcador un nombre significativo (p. ej., “Botón Enviar” o “Sección de resumen”).

  • Desplazamiento – Esto mueve la posición final del cursor en relación con el marcador encontrado. Introduce un número entero (positivo o negativo) para desplazar la posición de destino.

    Número positivo (p. ej., 2) mueve el cursor hacia adelante esa cantidad de párrafos después del marcador.

    Número negativo (p. ej., -1) mueve el cursor hacia atrás esa cantidad de párrafos antes del marcador.

    0 (predeterminado) aterriza exactamente en el marcador mismo.

Haz clic en Aceptar – tu marcador se guarda y está listo para usar.


Salta y haz clic como un profesional

Una vez que tengas marcadores configurados, puedes navegar por tu sitio sin esfuerzo:

  • Pulsa J, F, D o Z (solo) para saltar hacia adelante al siguiente marcador asignado a esa tecla.
  • Pulsa Mayús+J, Mayús+F, Mayús+D o Mayús+Z para saltar hacia atrás al marcador anterior.
  • Para marcadores de auto clic, pulsa Alt+J, Alt+C, Alt+X o Alt+Z – el cursor encontrará el siguiente marcador que coincida y hará clic automáticamente en el elemento de destino.

Cada salto lee el párrafo de destino en voz alta y mueve el cursor de revisión de pantalla a esa posición, para que siempre sepas dónde estás.


Gestionarlo todo

Siempre puedes pulsar dos veces Windows+F12 para abrir el Administrador de sitios. Desde allí puedes:

  • Editar el nombre mostrado, el patrón de URL o el tipo de coincidencia de un sitio.
  • Añadir, editar o eliminar marcadores para cualquier sitio.
  • Eliminar todos los marcadores de una vez.
  • Navegar de ida y vuelta entre la lista de sitios y la lista de marcadores.

El Administrador de marcadores (accesible desde el Administrador de sitios) te da una visión completa de todos los marcadores del sitio actual, con opciones rápidas para editar o eliminar cada uno.


Comparte tus marcadores

Todas las configuraciones de tus sitios y marcadores se almacenan en un sencillo archivo JSON para cada sitio. Estos archivos se encuentran en: 

%APPDATA%\nvda\ChaiChaimee\SiteMarker\site\

Puedes copiar estos archivos .json a otro ordenador o compartirlos con colegas. Solo colócalos en la misma carpeta en la otra máquina, y SiteMarker reconocerá los sitios y marcadores al instante – no se necesita configuración adicional. Esto facilita compartir configuraciones de navegación inteligentes entre equipos o dispositivos.


¿Por qué SiteMarker?

  • Ahorra tiempo – salta el maratón de flechas y ve directamente a donde necesitas.
  • Funciona en todas partes – admite Chrome, Edge, Firefox, Brave y navegadores basados en WebKit (incluyendo Safari en Windows).
  • Preparado para SPA – detecta automáticamente cambios de contenido en aplicaciones de una sola página (React, Angular, etc.) para que tus marcadores estén siempre actualizados.
  • Caché inteligente – coincidencia rápida incluso en páginas enormes.
  • Amigable con idiomas – maneja Unicode y coincidencia sin distinción de acentos para usuarios internacionales.
