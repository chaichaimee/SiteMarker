SiteMarker


Marquez votre chemin, sautez en toute simplicité


auteur : chai chaimee

URL : https://github.com/chaichaimee/SiteMarker



Description

Chaque jour, vous passez des heures sur les mêmes sites Web – à lire des articles, consulter des tableaux de bord, remplir des formulaires ou naviguer dans de longues listes. Pour passer d’une section importante à une autre, vous appuyez encore et encore sur les touches fléchées, à la recherche de ce paragraphe ou de ce bouton dont vous avez besoin. C’est répétitif, fatigant et cela gaspille votre temps précieux.

SiteMarker change cela. Il vous permet de placer des marqueurs permanents sur n’importe quel paragraphe ou élément d’une page Web. Une fois marqué, vous pouvez sauter directement à cet endroit avec une seule frappe – vers l’avant ou vers l’arrière, dans n’importe quel ordre. Fini le défilement interminable ou le marathon des flèches. Il suffit de marquer, sauter et avancer.


Raccourcis clavier

La touche principale pour toutes les actions est Windows+F12. Il s’agit d’une touche à appuis multiples – son comportement change selon le nombre de fois que vous l’appuyez rapidement (en moins de 0,4 seconde).

  • Appui simple (Windows+F12 une fois) – Si vous vous trouvez sur une ligne qui possède déjà un marqueur, la boîte de dialogue d’édition de ce marqueur s’ouvre. Si la ligne n’a pas encore de marqueur, la boîte de dialogue Ajouter un marqueur apparaît, pré‑remplie avec le texte du paragraphe actuel.
  • Double appui (Windows+F12 deux fois rapidement) – Ouvre le Gestionnaire de sites pour le site Web actuel. Si le site existe déjà, vous pouvez modifier ses paramètres ou gérer tous ses marqueurs. Si le site est nouveau, vous êtes invité à l’ajouter.

Premiers pas – Pas à pas

1. Ajoutez votre premier site

Lorsque vous ouvrez un site Web sur lequel vous souhaitez utiliser des marqueurs, appuyez deux fois sur Windows+F12. La boîte de dialogue Ajouter un site apparaît.

  • Nom affiché : Donnez à votre site un nom convivial (par exemple « Boîte de réception Gmail » ou « Actualités quotidiennes »).

  • Modèle d’URL : Saisissez l’adresse Web (ou une partie) qui identifie ce site.

  • Type de correspondance : Choisissez comment SiteMarker doit faire correspondre l’URL actuelle à votre modèle.

    Domaine uniquement – Correspond uniquement au domaine principal (par exemple, le modèle google.com fonctionne pour www.google.com, mail.google.com, mais pas pour google.co.uk).

    Inclure les sous‑domaines – Comme « Domaine uniquement », mais l’URL doit se terminer par votre modèle. Par exemple, le modèle example.com correspond à sub.example.com et news.example.com, mais pas à example.org.

    Contient la sous‑chaîne – Correspond si votre modèle apparaît n’importe où dans l’URL complète (y compris le chemin et la requête). Exemple : le modèle /search correspond à https://www.google.com/search?q=test.

    Correspondance exacte – L’URL complète doit être exactement égale à votre modèle (insensible à la casse). Utilisez‑le lorsque vous voulez des marqueurs uniquement pour une page spécifique, comme https://example.com/dashboard.

    Expression régulière – Pour les utilisateurs avancés. Écrivez un motif regex (par exemple https://.*\.example\.com/.*) pour correspondre à n’importe quelle page sous n’importe quel sous‑domaine de example.com.

Cliquez sur Enregistrer le site. Votre site est désormais enregistré et vous pouvez commencer à placer des marqueurs.


2. Placez votre premier marqueur

Naviguez jusqu’au paragraphe, à la ligne ou au bouton exact que vous voulez mémoriser. Ensuite, appuyez une fois sur Windows+F12. La boîte de dialogue Ajouter un marqueur s’ouvre.

  • Modèle – SiteMarker remplit automatiquement ce champ avec le texte actuellement focalisé. Vous pouvez le modifier si nécessaire. Ce texte est ce que SiteMarker recherchera.

  • Correspondance du modèle – Choisissez comment le modèle doit être comparé :

    Contient le texte – Trouve le modèle n’importe où dans le paragraphe (correspondance partielle).

    Paragraphe exact – Le paragraphe doit être exactement égal au modèle (correspondance texte intégral).

    Correspondance regex – Utilisez une expression régulière pour une correspondance avancée.

  • Mode – Décidez ce que fait ce marqueur :

    Sauter – Lorsque vous appuyez sur la touche de saut attribuée, le curseur se déplace vers ce marqueur. Vous pouvez attribuer l’une des quatre touches de saut : J, F, D ou Z.
    Appuyez sur la touche seule pour sauter vers l’avant jusqu’au prochain marqueur avec cette touche. Appuyez sur Maj+touche pour sauter vers l’arrière.

    Clic automatique – Au lieu de déplacer uniquement le focus, SiteMarker effectue automatiquement un clic sur l’élément lorsque vous appuyez sur la touche de clic attribuée. Les touches de clic disponibles sont Alt+J, Alt+C, Alt+X et Alt+Z.

  • Nom affiché – Donnez à votre marqueur un nom significatif (par exemple « Bouton Envoyer » ou « Section Résumé »).

  • Décalage – Cela déplace la position finale du curseur par rapport au marqueur trouvé. Saisissez un nombre entier (positif ou négatif) pour décaler la position d’atterrissage.

    Nombre positif (par exemple 2) déplace le curseur vers l’avant de ce nombre de paragraphes après le marqueur.

    Nombre négatif (par exemple -1) déplace le curseur vers l’arrière de ce nombre de paragraphes avant le marqueur.

    0 (par défaut) atterrit exactement sur le marqueur lui‑même.

Cliquez sur OK – votre marqueur est enregistré et prêt à être utilisé.


Sauter et cliquer comme un pro

Une fois vos marqueurs configurés, vous pouvez naviguer sur votre site sans effort :

  • Appuyez sur J, F, D ou Z (seul) pour sauter vers l’avant jusqu’au prochain marqueur attribué à cette touche.
  • Appuyez sur Maj+J, Maj+F, Maj+D ou Maj+Z pour sauter vers l’arrière jusqu’au marqueur précédent.
  • Pour les marqueurs à clic automatique, appuyez sur Alt+J, Alt+C, Alt+X ou Alt+Z – le curseur trouvera le prochain marqueur correspondant et cliquera automatiquement sur l’élément cible.

Chaque saut lit le paragraphe cible à voix haute et déplace le curseur de revue d’écran à cette position, afin que vous sachiez toujours où vous êtes.


Gérer tout

Vous pouvez toujours appuyer deux fois sur Windows+F12 pour ouvrir le Gestionnaire de sites. À partir de là, vous pouvez :

  • Modifier le nom affiché, le modèle d’URL ou le type de correspondance d’un site.
  • Ajouter, modifier ou supprimer des marqueurs pour n’importe quel site.
  • Effacer tous les marqueurs d’un coup.
  • Naviguer entre la liste des sites et la liste des marqueurs.

Le Gestionnaire de marqueurs (accessible depuis le Gestionnaire de sites) vous donne une vue d’ensemble complète de tous les marqueurs du site actuel, avec des options rapides pour modifier ou supprimer chacun d’eux.


Partagez vos marqueurs

Toutes les configurations de vos sites et marqueurs sont stockées dans un simple fichier JSON pour chaque site. Ces fichiers se trouvent dans : 

%APPDATA%\nvda\ChaiChaimee\SiteMarker\site\

Vous pouvez copier ces fichiers .json sur un autre ordinateur ou les partager avec des collègues. Placez‑les simplement dans le même dossier sur l’autre machine, et SiteMarker reconnaîtra immédiatement les sites et les marqueurs – aucune configuration supplémentaire n’est nécessaire. Cela facilite le partage de configurations de navigation intelligentes entre équipes ou appareils.


Pourquoi SiteMarker ?

  • Gagnez du temps – évitez le marathon des flèches et sautez directement là où vous devez aller.
  • Fonctionne partout – prend en charge Chrome, Edge, Firefox, Brave et les navigateurs basés sur WebKit (y compris Safari sous Windows).
  • Prêt pour les SPA – détecte automatiquement les changements de contenu dans les applications monopages (React, Angular, etc.) afin que vos marqueurs restent toujours à jour.
  • Mise en cache intelligente – correspondance rapide même sur de grandes pages.
  • Respectueux des langues – gère Unicode et la correspondance insensible aux accents pour les utilisateurs internationaux.

Commencez à marquer vos sites préférés dès aujourd’hui – et transformez chaque site Web en un terrain de jeu de navigation personnel.
