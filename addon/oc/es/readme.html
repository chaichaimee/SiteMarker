SiteMarker

Marque su camino, salte con facilidad

autor: chai chaimee

Introducción

SiteMarker es un complemento de NVDA que le permite guardar y acceder rápidamente a secciones de uso frecuente en cualquier sitio web. Con solo unas pocas pulsaciones, puede marcar párrafos, botones o controles importantes y saltar a ellos al instante – sin más desplazamientos interminables ni búsquedas.


Características

  • Coloque marcadores permanentes en cualquier elemento de una página web.
  • Salte hacia adelante o hacia atrás a los marcadores usando las teclas J, F, D, Z.
  • Auto‑clic: los marcadores pueden activar automáticamente botones, enlaces o elementos de formulario.
  • Administre sitios y marcadores a través del Administrador de sitios y el Administrador de marcadores.
  • Comparta sus configuraciones de marcadores entre dispositivos mediante archivos JSON portables.
  • Funciona con Chrome, Edge, Firefox, Brave y navegadores basados en WebKit.
  • Compatible con aplicaciones de una sola página (SPA) con detección automática de cambios de contenido.
  • Métodos de coincidencia flexibles: texto, exacto o expresiones regulares.

Beneficios

  • Ahorre tiempo – elimine el desplazamiento repetitivo y la navegación con flechas.
  • Reduzca la fatiga – saltos rápidos con una sola tecla mantienen sus manos cómodas.
  • Aumente la productividad – cree accesos directos personalizados para sus tareas más frecuentes.
  • Manténgase orientado – la retroalimentación por voz en cada salto le indica su ubicación.

¿Por qué usarlo?

Cada día visita los mismos sitios web – leyendo, rellenando formularios, revisando paneles. Moverse entre secciones clave puede ser tedioso y agotador. SiteMarker resuelve esto permitiéndole colocar marcadores exactamente en los lugares que necesita. Sin más búsquedas; solo marque, salte y concéntrese en lo importante.



Primeros pasos – Paso a paso


1. Agregar su primer sitio

Cuando abra un sitio web donde desee usar marcadores, pulse dos veces Windows+F12. Aparecerá el diálogo Agregar sitio.

  • Nombre para mostrar: Asigne un nombre amigable (p. ej., “Bandeja de Gmail” o “Noticias diarias”).

  • Patrón de URL: Introduzca la dirección web (o parte de ella) que identifica a este sitio.

  • Tipo de coincidencia: Elija cómo SiteMarker debe comparar la URL actual con su patrón.

    Solo dominio – Coincide solo con el dominio principal (p. ej., google.com funciona para www.google.com y mail.google.com, pero no para google.co.uk).

    Incluir subdominios – La URL debe terminar con su patrón. Ejemplo: example.com coincide con sub.example.com y news.example.com, pero no con example.org.

    Contener subcadena – Coincide si su patrón aparece en cualquier parte de la URL completa (incluyendo ruta y consulta). Ejemplo: /search coincide con https://www.google.com/search?q=test.

    Coincidencia exacta – La URL completa debe ser exactamente igual a su patrón (sin distinguir mayúsculas/minúsculas). Úselo cuando quiera marcadores solo para una página específica como https://example.com/dashboard.

    Expresión regular – Para usuarios avanzados. Escriba un patrón regex (p. ej., https://.*\.example\.com/.*) para coincidir con cualquier página bajo cualquier subdominio de example.com.

Haga clic en Guardar sitio. Su sitio queda registrado y puede empezar a colocar marcadores.


2. Colocar su primer marcador

Navegue hasta el párrafo, línea o botón exacto que desea recordar. Luego pulse una vez Windows+F12. Se abrirá el diálogo Agregar marcador.

  • Patrón: SiteMarker lo rellena automáticamente con el texto que tiene el foco. Puede editarlo si es necesario. Este texto es el que SiteMarker buscará.

  • Comparación de patrón: Elija cómo se debe comparar el patrón:

    Contiene texto – Encuentra el patrón en cualquier lugar dentro del párrafo (coincidencia parcial).

    Párrafo exacto – El párrafo debe ser igual al patrón (coincidencia de texto completo).

    Coincidencia regex – Use una expresión regular para búsquedas avanzadas.

  • Modo: Decida qué hace este marcador:

    Saltar – Cuando pulse la tecla de salto asignada, el cursor se moverá a este marcador. Puede asignar una de cuatro teclas: J, F, D o Z. Pulse la tecla sola para saltar adelante al siguiente marcador con esa tecla. Pulse Mayús+tecla para saltar atrás.

    Auto clic – En lugar de solo mover el foco, SiteMarker hará un clic automáticamente en el elemento cuando pulse la tecla de clic asignada. Las teclas de clic disponibles son Alt+J, Alt+C, Alt+X y Alt+Z.

  • Nombre para mostrar: Asigne un nombre significativo (p. ej., “Botón Enviar” o “Sección de resumen”).

  • Desplazamiento (offset): Desplaza la posición final del cursor en relación con el marcador encontrado. Introduzca un número entero (positivo o negativo) para ajustar la posición de destino.

    Número positivo (p. ej., 2) mueve el cursor adelante esa cantidad de párrafos después del marcador.

    Número negativo (p. ej., -1) mueve el cursor atrás esa cantidad de párrafos antes del marcador.

    0 (predeterminado) coloca el cursor exactamente sobre el marcador.

Haga clic en Aceptar – su marcador queda guardado y listo para usar.



Saltar y hacer clic como un profesional

Una vez que tenga marcadores configurados, puede navegar por su sitio sin esfuerzo:

  • Pulse J, F, D o Z (solo) para saltar adelante al siguiente marcador asignado a esa tecla.
  • Pulse Mayús+J, Mayús+F, Mayús+D o Mayús+Z para saltar atrás al marcador anterior.
  • Para marcadores de auto clic, pulse Alt+J, Alt+C, Alt+X o Alt+Z – el cursor buscará el siguiente marcador coincidente y hará clic automáticamente en el elemento objetivo.

Cada salto lee el párrafo objetivo en voz alta y mueve el cursor de revisión de pantalla a esa posición, para que siempre sepa dónde está.


Gestionarlo todo

Siempre puede abrir el Administrador de sitios pulsando dos veces Windows+F12. Desde allí podrá:

  • Editar el nombre para mostrar, el patrón de URL o el tipo de coincidencia de un sitio.
  • Agregar, editar o eliminar marcadores para cualquier sitio.
  • Eliminar todos los marcadores de una vez.
  • Navegar entre la lista de sitios y la lista de marcadores.

El Administrador de marcadores (accesible desde el Administrador de sitios) le ofrece una visión completa de todos los marcadores del sitio actual, con opciones rápidas para editar o eliminar cada uno.


Compartir sus marcadores

Todas las configuraciones de sitios y marcadores se almacenan en simples archivos JSON por sitio. Estos archivos se encuentran en: 

%APPDATA%\nvda\ChaiChaimee\SiteMarker\site\

Puede copiar estos archivos .json a otro ordenador o compartirlos con colegas. Simplemente colóquelos en la misma carpeta en la otra máquina, y SiteMarker reconocerá los sitios y marcadores al instante – sin necesidad de configuración adicional. Así puede compartir configuraciones de navegación inteligentes entre equipos o dispositivos.
