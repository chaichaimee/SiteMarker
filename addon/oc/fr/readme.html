SiteMarker

Marquez votre chemin, sautez en toute simplicité

auteur : chai chaimee

Introduction

SiteMarker est un module complémentaire NVDA qui vous permet d’enregistrer et d’accéder rapidement aux sections les plus utilisées de n’importe quel site Web. En quelques frappes, vous pouvez marquer des paragraphes, des boutons ou des contrôles importants et y revenir instantanément – fini les défilements interminables et les recherches fastidieuses.


Fonctionnalités

  • Placez des marqueurs permanents sur n’importe quel élément d’une page Web.
  • Sautez vers l’avant ou vers l’arrière vers les marqueurs à l’aide des touches J, F, D, Z.
  • Clic automatique : les marqueurs peuvent activer automatiquement des boutons, des liens ou des éléments de formulaire.
  • Gérez les sites et les marqueurs via le Gestionnaire de sites et le Gestionnaire de marqueurs.
  • Partagez vos configurations de marqueurs entre appareils via des fichiers JSON portables.
  • Fonctionne avec Chrome, Edge, Firefox, Brave et les navigateurs basés sur WebKit.
  • Prend en charge les applications mono‑page (SPA) avec détection automatique des changements de contenu.
  • Méthodes de correspondance flexibles : texte, exact ou expressions régulières.

Avantages

  • Gagnez du temps – éliminez les défilements répétitifs et la navigation par flèches.
  • Réduisez la fatigue – des sauts rapides avec une seule touche préservent vos mains.
  • Augmentez votre productivité – créez des raccourcis personnalisés pour vos tâches fréquentes.
  • Gardez le cap – un retour vocal à chaque saut vous indique votre position.

Pourquoi l’utiliser ?

Chaque jour, vous visitez les mêmes sites Web – pour lire, remplir des formulaires, consulter des tableaux de bord. Se déplacer entre les sections clés peut être fastidieux et fatigant. SiteMarker résout ce problème en vous permettant de placer des marqueurs exactement aux endroits dont vous avez besoin. Plus de recherches inutiles ; il suffit de marquer, de sauter et de se concentrer sur l’essentiel.



Premiers pas – Étape par étape


1. Ajouter votre premier site

Lorsque vous ouvrez un site Web sur lequel vous souhaitez utiliser des marqueurs, appuyez deux fois sur Windows+F12. La boîte de dialogue Ajouter un site s’affiche.

  • Nom d’affichage : Donnez un nom convivial (ex. « Boîte de réception Gmail » ou « Actualités quotidiennes »).

  • Motif d’URL : Saisissez l’adresse Web (ou une partie) qui identifie ce site.

  • Type de correspondance : Choisissez comment SiteMarker doit comparer l’URL actuelle à votre motif.

    Domaine uniquement – Correspond uniquement au domaine principal (ex. google.com fonctionne pour www.google.com et mail.google.com, mais pas pour google.co.uk).

    Inclure les sous‑domaines – L’URL doit se terminer par votre motif. Exemple : example.com correspond à sub.example.com et news.example.com, mais pas à example.org.

    Contient une sous‑chaîne – Correspond si votre motif apparaît n’importe où dans l’URL complète (y compris le chemin et la requête). Exemple : /search correspond à https://www.google.com/search?q=test.

    Correspondance exacte – L’URL complète doit être exactement égale à votre motif (insensible à la casse). Utilisez‑le lorsque vous voulez des marqueurs uniquement pour une page spécifique comme https://example.com/dashboard.

    Expression régulière – Pour les utilisateurs avancés. Écrivez un motif regex (ex. https://.*\.example\.com/.*) pour correspondre à n’importe quelle page sous n’importe quel sous‑domaine de example.com.

Cliquez sur Enregistrer le site. Votre site est maintenant enregistré et vous pouvez commencer à placer des marqueurs.


2. Placer votre premier marqueur

Accédez au paragraphe, à la ligne ou au bouton exact que vous souhaitez mémoriser. Appuyez ensuite une fois sur Windows+F12. La boîte de dialogue Ajouter un marqueur s’ouvre.

  • Motif : SiteMarker le remplit automatiquement avec le texte actuellement sous le focus. Vous pouvez le modifier si nécessaire. Ce texte est ce que SiteMarker recherchera.

  • Correspondance du motif : Choisissez comment le motif doit être comparé :

    Contient du texte – Recherche le motif n’importe où dans le paragraphe (correspondance partielle).

    Paragraphe exact – Le paragraphe doit être identique au motif (correspondance de texte entier).

    Correspondance regex – Utilisez une expression régulière pour une recherche avancée.

  • Mode : Déterminez ce que fait ce marqueur :

    Sauter – Lorsque vous appuyez sur la touche de saut attribuée, le curseur se déplace vers ce marqueur. Vous pouvez attribuer l’une des quatre touches : J, F, D ou Z. Appuyez sur la touche seule pour sauter vers l’avant jusqu’au prochain marqueur associé à cette touche. Appuyez sur Maj+touche pour sauter vers l’arrière.

    Clic automatique – Au lieu de déplacer uniquement le focus, SiteMarker effectue un clic automatique sur l’élément lorsque vous appuyez sur la touche de clic attribuée. Les touches de clic disponibles sont Alt+J, Alt+C, Alt+X et Alt+Z.

  • Nom d’affichage : Donnez à votre marqueur un nom significatif (ex. « Bouton Envoyer » ou « Section Résumé »).

  • Décalage : Déplace la position finale du curseur par rapport au marqueur trouvé. Saisissez un nombre entier (positif ou négatif) pour ajuster la position d’atterrissage.

    Nombre positif (ex. 2) déplace le curseur vers l’avant de ce nombre de paragraphes après le marqueur.

    Nombre négatif (ex. -1) déplace le curseur vers l’arrière de ce nombre de paragraphes avant le marqueur.

    0 (par défaut) positionne le curseur exactement sur le marqueur.

Cliquez sur OK – votre marqueur est enregistré et prêt à l’emploi.



Sauter et cliquer comme un pro

Une fois vos marqueurs configurés, vous pouvez naviguer sur votre site sans effort :

  • Appuyez sur J, F, D ou Z (seul) pour sauter vers l’avant jusqu’au prochain marqueur attribué à cette touche.
  • Appuyez sur Maj+J, Maj+F, Maj+D ou Maj+Z pour sauter vers l’arrière jusqu’au marqueur précédent.
  • Pour les marqueurs de clic automatique, appuyez sur Alt+J, Alt+C, Alt+X ou Alt+Z – le curseur recherchera le prochain marqueur correspondant et cliquera automatiquement sur l’élément cible.

Chaque saut énonce le paragraphe cible et déplace le curseur de révision de l’écran à cette position, afin que vous sachiez toujours où vous vous trouvez.


Gérer tout

Vous pouvez à tout moment ouvrir le Gestionnaire de sites en appuyant deux fois sur Windows+F12. Depuis celui‑ci, vous pouvez :

  • Modifier le nom d’affichage, le motif d’URL ou le type de correspondance d’un site.
  • Ajouter, modifier ou supprimer des marqueurs pour n’importe quel site.
  • Effacer tous les marqueurs en une seule fois.
  • Naviguer entre la liste des sites et la liste des marqueurs.

Le Gestionnaire de marqueurs (accessible depuis le Gestionnaire de sites) vous donne une vue d’ensemble de tous les marqueurs du site actuel, avec des options rapides pour les modifier ou les supprimer.


Partager vos marqueurs

Toutes vos configurations de sites et de marqueurs sont stockées dans de simples fichiers JSON par site. Ces fichiers se trouvent dans : 

%APPDATA%\nvda\ChaiChaimee\SiteMarker\site\

Vous pouvez copier ces fichiers .json sur un autre ordinateur ou les partager avec des collègues. Placez‑les simplement dans le même dossier sur l’autre machine, et SiteMarker reconnaîtra instantanément les sites et les marqueurs – aucune configuration supplémentaire n’est nécessaire. Cela permet de partager facilement des configurations de navigation intelligentes entre équipes ou appareils.
